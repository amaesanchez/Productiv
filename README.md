# Productiv

Swedish to-do app with features for prioritizing tasks and retrieving randomized quotes.

## Local Setup

1. Install dependencies and run app.

    ```
    npm install
    npm start
    ```
Open [localhost:3000](https://localhost:3000) in browser to view.
